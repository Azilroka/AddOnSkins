<h4 class="Overlord">Tukui & ElvUI Skins Package version 3 is out!</h4>

<p class="VIP">This new version is a completely new addon ready for your use.

<b>The following has changed from version 2.1D:</b>

<ul>
<li> A new addon name, <em class="Premium">Tukui_ElvUI_Skins</em>.  You should remove the old <em>Tukui_UIPackages_Skins</em> before installing this as they will conflict.</li>
<li> New backends for Tukui and ElvUI written by <span class="Moderator">Sortokk</span> designed to lower the memory footprint of the skins by not creating unnecessary frames.</li>
<li>Options for skins under ElvUI are now handled using the standard ElvUI configuration system:
	<em class="Duelist">ElvUI Config > Skins > AddOns > Embed Settings
  </em></li>
</ul></p>

<h5 class="Member">Commands</h5>
<ul>
<li class="Member"><b>/embed</b> : <em class="VIP">Open the Embed Window</em></li>
<li class="Member"><b>/es | /embedskada</b> : <em class="VIP">Embed Skada</em></li>
<li class="Member"><b>/skadabackdrop</b> : <em class="VIP">Enable/Disable Skada's Backdrop</em></li>
<li class="Member"><b>/er | /embedrecount</b> : <em class="VIP">Embed Recount</em></li>
<li class="Member"><b>/recountbackdrop</b> : <em class="VIP">Enable/Disable Recount's Backdrop</em></li>
<li class="Member"><b>/skinoptions</b> : <em class="VIP">Opens the Skin Options</em></li>
</ul>
